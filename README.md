##  À propos

Simulation d'une **page de blog complète** en une seule page HTML. L'article « 10 astuces de développement web pour les débutants » intègre commentaires, barre latérale et tableau comparatif d'outils.

> 💡 Deux `<header>` et deux `<footer>` : un pour le site, un pour l'article — c'est normal en HTML sémantique.

##  Fonctionnalités

- ✅ **Article structuré** — `<article>`, `<section>`, `<figure>`, `<time>`
- ✅ **Commentaires** — `ol > li > article` avec auteurs et dates
- ✅ **Sidebar** — articles récents + catégories (sans wrapper `<div>`)
- ✅ **Tableau comparatif** — VS Code, DevTools, Git, etc.
- ✅ **Balises avancées** — `<blockquote>`, `<cite>`, `<abbr>`, `<mark>`

##  Prérequis

- Navigateur web moderne
- *(Optionnel)* [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer)

##  Installation

```bash
git clone https://github.com/Chal-B/article_de_blog_structure.git
cd article_de_blog_structure
```

Ouvrir **`article.html`** dans le navigateur.

##  Utilisation

Projet statique — ouvrir `article.html` et naviguer vers Article, Commentaires, Articles récents ou Catégories.
